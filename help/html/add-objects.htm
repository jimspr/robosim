(add-objects {object}+)

Returns a new object that is the union of all of the specified objects. This copy is not affected by any changes made later to the component objects.

(setq stack (add-objects box1 box2 box3)) => creates a single object that consists of box1, box2, and box3. This occurs by combining the display lists. Therefore, there can not be multiple attributes, such as color, for the object.
