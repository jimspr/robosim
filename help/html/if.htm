(if test then [else])

If the result of evaluating the test form is not nil, the then form is evaluated and its value returned. Otherwise, the else form is evaluated and its value returned. If the else form is omitted, then nil is returned.

(setq x 1) => 1
(if (= x 1) (+ x 1) (- x 1)) => 2
(setq x 5) => 5
(if (= x 1) (+ x 1) (- x 1)) => 4
