(set-light light flag [:position (x y z)] [:ambient ambient] [:diffuse diffuse] [:specular specular])

Sets the lighting values for the light specified by light, which must be in the range of 0 to 7. Flag controls whether the light is on(t) or off(nil). Optional parameters allow you to set the position of the light as well as ambient, specular, and diffuse components. The ambient, diffuse, and specular parameters are lists of three values in the range from -1.0 to 1.0.
