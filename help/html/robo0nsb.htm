(make-prismatic-link minvalue maxvalue {object transform*})

Creates a prismatic robot link composed of the specified objects. The object is transformed by the specified transform and added to the link. The object is then transformed back, which leaves it unchanged. The minvalue specifies the minimum value of the joint, and maxvalue specifies the maximum value of the joint. If minvalue is nil then it takes on the most negative floating point number. If maxvalue is nil then it takes on the most positive floating point number.
