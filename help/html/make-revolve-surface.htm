(make-revolve-surface object [facets])

Creates an object by revolving the object around the z axis. The number of facets is specified by facets. If not specified, then the number specified in set-num-facets is used.

(make-revolve-surface
    (make-primitive (list
        #(0 0 0)
        #(3 0 0)
        #(5 0 5)
        #(3 0 10)
        #(0 0 10)
    ))
    20
)
