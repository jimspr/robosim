(let[*] ({var | (var value)}* {form}*)

Evaluates the forms with the variables specified by var bound to the values specified by value. The bindings occur in parallel; all values are evaluated before any bindings take place. LET* performs the bindings serially. The result of the last form is the value returned. The following form illustrates the structure a little more clearly.

(let
        (
            (var1 value1)
            (var2 value2)
            ...
            (varn valuen)
        )
        form1
        form2
        ...
        formn
)
(let
    (
        (x 4)
        (x2 16)
    )
    (+ x x2)
) => 20
