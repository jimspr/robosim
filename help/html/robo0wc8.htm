ROBOSIM II Version 1.4

This help file describes the features and language of ROBOSIM II. The programming language built into ROBOSIM II is a subset of Common LISP. The tutorial gives an overview of the LISP language and the extensions provided for ROBOSIM. The What's New section covers what has been added since version 1.0.
