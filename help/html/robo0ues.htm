(make-serial-agent [:classic] {link transform*})

This command creates a serial manipulator of links. Each transform relates the position of that link to the previous link. If :classic is present then the link's motion is defined by the preceding link's z axis. This is consistent with the way R.P. Paul [PAUL81] defines links. If :classic is not present then the link's z axis defines the axis of motion for that link. This is consistent with J.J. Craig. [CRAIG89]
