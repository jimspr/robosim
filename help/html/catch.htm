(catch tag {form}*)

The form tag is evaluated to produce an object that names the catch. The forms are executed and the value of the last one is returned. However, if a throw occurs and the tag specified by the throw is eq to this tag then the value specified in the throw is returned.

(catch 'collision
    (loop
        (move-robot)
        (if
            (check-collision rob1 ob1)
            (throw 'collision nil)
        )
    )
)
