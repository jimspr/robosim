(defun name lambda-list {form}*)

Defines a function called name. For example,

(defun distance (x y)
        (sqrt (+ (* x x) (* y y)))
)

This defines a function called distance that takes two parameters and returns the distance to the origin. A function can be declared with no parameters:

(defun add1tox nil
        (setq x (+ x 1))
)

This function adds one to the global variable x and return the value.
