(tref transform row col)

Returns the specified element of a transformation matrix. The row and column indexes are zero based. Valid values for row and col are 0 - 3. A transformation is in the following form.

Nx
 
Sx
 
Ax
 
Px
 
Ny
 
Sy
 
Ay
 
Py
 
Nz
 
Sz
 
Az
 
Pz
 
0
 
0
 
0
 
1
 

(aref (scale 2 3 4) 1 1) => 3    ;Sy element
(aref (scale 2 3 4) 1 2) => 0    ;Ay element
