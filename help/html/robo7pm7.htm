(do[*]
( {var | (var [init [step]])}*)
(end-test {result}*)
{statement}*
)

All of the init forms are evaluated and then bound to the corresponding var. do evaluates all of the init forms before binding. do* evaluates and binds left to right. The end-test is evaluated. If the end-test returns nil then the statements are evaluated in left to right order. Then the step forms are evaluated and the variables are changed. The end-test form is then evaluated. This process continues until the end-test form returns a non-nil value. When this occurs, the result forms are evaluated and the result of the last result form is returned. If there are no result forms, nil is returned.

(do
    (
        (x 1 (+ x 1))
        (y 0)
    )
    ( (> x 10) y)
    (setq y (+ y x))
) => 55
