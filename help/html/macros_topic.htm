Macros

Macros are similar to special forms and can be implemented as such. In this implementation, macros are implemented as special forms. Although the Common LISP standard says that it should be possible to get the expansion of a macro, it is not possible in this implementation. As in special forms, arguments are not evaluted before the macro is invoked.
