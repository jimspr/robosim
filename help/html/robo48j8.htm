(set-object object [:color color] [:ambient ambient] [:diffuse diffuse] [:specular specular])

Sets the object to the given color. A color is list of three elements representing the red, green, and blue components. Each component can have a value between 0 and 255, inclusive. The ambient, diffuse, and specular parameters are lists of three values in the range from -1.0 to 1.0. The return value is object. For example, the following command sets the object mybox to the color red. The :color and :ambient parameters actually are two different ways to specify the same.

(set-object mybox :color '(255 0 0)) => #<SimOb:xxxx>
(set-object mybox :ambient '(1.0 0.0 0.0)) => #<SimOb:xxxx>
