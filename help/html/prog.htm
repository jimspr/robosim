(prog[*] ( {var | (var [init ])}*) {statement}*)

All of the init forms are evaluated and then bound to the corresponding var. prog evaluates all of the init forms before binding. prog* evaluates and binds left to right. The statements are evaluated in left to right order, returning the result of the last statement. Also, there is an implicit block created with the name nil. A return can be used to break out anywhere within the prog.

(prog
    (
        (x 1)
        (y 2)
    )
    (+ x y)
) => 3
