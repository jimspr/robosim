(apply fn arg {argument}*)

The function fn is called with an argument list of all arguments passed and if the last argument is a list appending those to all the other arguments.


	(setq stepfunc (function -)) => #<subr:xxxxxxxx>
	(apply stepfunc '(4)) => -4
	(apply (function +) '(3 5)) => 8
	(apply (function +) 1 2 3 '(4 5 6)) => 21
	(apply #'+ '(3 5)) => 8
