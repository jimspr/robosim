(ask-dialog function titlestring [{question-string initial-value}*])

Creates a dialog box with the title titlestring. The parameter function can be either nil or a function. It it is a function, then any time one of the values changes, the function is called. A list of questions is presented. Question-string is the text of the question. The initial-value must be an integer, a floating point number, a string, or a symbol. ASK-DIALOG returns a list of the values given by the user. If the user hits escape while in the dialog box, or presses cancel, then nil is returned.

(ask-dialog nil
    "Get Position"
    "X Pos" 10.0
    "Y Pos" 20.0
    "Z Pos" 3.5
)
