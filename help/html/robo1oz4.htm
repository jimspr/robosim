(loop {form}*)

Evaluates each form from left to right. When the last form has been evaluated, the first form is evaluated again, and the cycle repeats. The only way to break out of the loop is with return. loop creates an implicit block with the name of nil.

(setq x 0)
(setq y 0)
(loop
    (setq x (+ x 1))
    (setq y (+ y x))
    (if
        (> x 10)
        (return y)
    )
) => 55
