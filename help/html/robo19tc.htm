(equalp x y)

Same as for equal except for the following cases. If they are characters, a case-insensitive comparison is made. Numbers do not have to be the same type to be equalp. Objects that have components, such as lists, strings, and vectors, are equalp if the components are equalp.

(equalp '(a b) '(a b)) => t
(equalp 3 3.0) => t
(equalp '(1 2 3) '(1.0 2.0 3.0)) => t
