(make-composite {object transform*})

Creates a composite object composed of the specified objects. The object is transformed by the specified transform and added to the link. The object is then transformed back, which leaves it unchanged. This function is implemented exactly the same as make-fixed-link.

(make-composite
    (make-cylinder 2.7 9.16) (translate 0 0 -3.88)
    (make-cylinder 3.5 1) (translate 0 0 6.39)
    (make-cylinder 3 1) (translate 0 0 5.39)
)
