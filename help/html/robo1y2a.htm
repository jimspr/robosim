(mapcar fn {lists}+)

The function fn is called first with the first element of each list, then the second element, and so on until the lists are exhausted. The return value is a list formed from the results of the successive function calls.

(mapcar #'+ '(1 2 3 4) '(100 200 300 400)) =>
    (101 202 303 404)
(mapcar #'cons '(1 2 3 4) '(100 200 300 400)) =>
    ((1 . 100) (2 . 200) (3 . 300) (4 . 400))
