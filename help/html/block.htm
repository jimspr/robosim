(block name {form}*)

Executes the forms sequentially, returning the result of the last form. However, if a return or return-from with the same name appears, then the value specified in the return is immediately returned.

(setq x 5) => 5
(block myblock
    (if
        (= x 2)
        (return-from myblock nil)
    )
    (setq x (1+ x))
) => 6
(setq x 2) => 2
(block myblock
    (if
        (= x 2)
        (return-from myblock nil)
    )
    (setq x (1+ x))
) => nil
