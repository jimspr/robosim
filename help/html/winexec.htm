(winexec string {args}*)

Executes the command that is constructed from string and args. Returns t if successful, otherwise it returns nil. (winexec "notepad" "startup.lsp") will not work because no spaces are inserted automatically. This command is equivalent to (winexec "notepadstartup.lsp"), which won't work.

(winexec "notepad startup.lsp") => t
(winexec "notepad " "startup.lsp") => t
(setq x 10)
(winexec "notepad " "test" x ".lsp") => t
This is equivalent to (winexec "notepad test10.lsp")
