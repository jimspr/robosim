(setq {var form}*)

Simple variable assignment. Each form is evaluated and the result stored in the specified var. (setq a 1 b (+ 1 2)) puts 1 in a and 3 in b. These are evaluated sequentially, so the expression (setq a 2 b (+ a 2)) results in the value of b being 4. psetq performs a parallel setq. That is, all forms are evaluated before any assignments take place.

(setq x 2) => 2
(setq x (+ x 1)) => 3
(setq x 1 y (+ x 1) z (+ y 1)) => 3
