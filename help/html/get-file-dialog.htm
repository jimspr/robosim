(get-file-dialog [defext filetypes])

This command pops up a dialog box from which a user can select a file. It returns a string containing the filename the user picked or nil if the user pressed the CANCEL button. defext is the default extension when the dialog box is opened. filetypes are the available types listed in the dialog box.

(get-file-dialog "LISP Files (*.lsp)|*.lsp"
    "LISP Files (*.lsp)|*.lsp|All Files (*.*)|*.*")
