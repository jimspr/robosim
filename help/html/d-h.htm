(d-h theta dz da alpha)

Returns a transformation corresponding to the given Denavit-Hartenberg parameters where the angles are in degrees.

(d-h theta dz da alpha) is equivalent to  
(matmult
    (rotatez theta)
    (translate da 0 dz)
    (rotatex alpha)
)
(d-h 90 2.5 8 0) =>
    #<mat44 0 -1 0 0 1 0 0 8 0 0 1 2.5 0 0 1>
