(cond {(test {form}*)}*)

The tests are evaluated sequentially. As soon as one of them returns a non-nil value, then the forms associated with that test are evaluated in left to right order and the result of the last one is returned. If there are no forms associated with a test, then the result of the test is returned. If none of the tests return non-nil values then nil is returned.

(setq x 2) => 1
(cond
    ( (= x 0) "zero")
    ( (< x 0) "negative")
    ( t "positive")
) => positive
