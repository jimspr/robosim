(make-primitive {polygon}*)

Creates a set of polygons using the given polygons. A polygon is a list of points. The last point of a polygon must be the same as the first point. There must be at least three points in a polygon. All points should lie in the same plane. The syntax for a point is #(x y z [nx ny nz]), where x, y, and z are the coordinates of a point and nx, ny, and nz are the normal for that point. The normal is optional and is only used for smooth (Gouraud) shading.

(make-polygon
    '(
        #(0 0 0)
        #(0 10 0)
        #(10 10 0)
        #(10 0 0)
        #(0 0 0)
    )
)
