(throw tag result)

The tag is evaluated to produce an object that names the tag. This form transfers control to the catch form with the same tag as that produced by tag. The result form is evaluated and is the value given to the catch.

(catch 'collision
    (loop
        (move-robot)
        (if
            (check-collision rob1 ob1)
            (throw 'collision nil)
        )
    )
)
