(make-extrude-surface object height)

Creates an extruded object of the given height.

(make-extrude-surface
        (make-primitive
            '(
                #(0 0 0)
                #(0 10 0)
                #(10 10 0)
                #(10 0 0)
                #(0 0 0)
            )
        )
        10
)
